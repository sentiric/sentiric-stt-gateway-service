# 🧠 AI SYSTEM INSTRUCTIONS FOR STT-GATEWAY-SERVICE
1. **Teknoloji:** Rust 1.84+ (Tokio, Tonic).
2. **Medya İşleme Yasağı:** Ses verisinin (audio chunk) içeriğini analiz edemez, modifiye edemez veya diske (WAV/MP3) kaydedemez. Sadece stream proxy (Zero-Copy) görevi görür.
3. **mTLS Strict:** Upstream (Whisper) motorlarına bağlanırken Insecure gRPC kullanılamaz. Sertifika yoksa servis `panic!` atmalıdır.
4. **Tenant İzolasyonu:** İsteklerde `x-tenant-id` yoksa, çağrı derhal `grpc::Status::invalid_argument` ile reddedilmelidir.
